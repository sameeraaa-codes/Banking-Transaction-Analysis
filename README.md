# Banking-Transaction-Analysis
Banking transaction data analysis using Python, Pandas, NumPy and Matplotlib to identify customer behavior and transaction trends.

## Description
This project analyzes banking transaction data to understand customer behavior and transaction trends. 
It includes data cleaning, preprocessing, and visualizations to extract meaningful insights from the dataset.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib
- Jupyter Notebook

## Objective
- Perform data cleaning and preprocessing on banking transaction data
- Conduct monthly transaction analysis
- Compare debit vs credit transactions
- Identify customer spending patterns
- Visualize trends and insights for decision-making

## Features
- Monthly transaction summaries
- Debit vs Credit analysis
- Customer behavior analysis
- Statistical analysis (mean, median, mode, etc.)
- Visualizations using Matplotlib

## Dataset
The project uses a banking transaction dataset (CSV) containing transaction details such as:
- Date of transaction
- Type (Debit/Credit)
- Amount
- Customer details
1. Open `Banking_Transaction_Analysis.ipynb` in Jupyter Notebook
2. Load the CSV dataset
3. Run all cells to see the analysis, visualizations, and insights
